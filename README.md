# 🧩 Puzzle Game - Fullstack Taquin

Bienvenue dans mon projet de **Jeu de Taquin (Sliding Puzzle)** entièrement personnalisé. C'est une application Fullstack développée avec la stack **React, Node.js et MySQL**.

## ✨ Pourquoi ce projet est unique ?
Contrairement à un simple puzzle, cette application offre une expérience communautaire complète :
- **🎮 Jeu de Taquin Interactif :** Une logique de déplacement de bloc unique (Sliding Puzzle) avec calcul du temps de jeu.
- **🖼️ Galerie Communautaire :** Découvrez les photos utilisées par les autres joueurs et explorez leurs créations.
- **🏆 Système de Classement :** Un "Leaderboard" en temps réel affichant les joueurs ayant réalisé les temps les plus courts.

## 🛠️ Stack Technique
- **Frontend :** React.js (Hooks, Context API)
- **Backend :** Node.js & Express
- **Base de données :** MySQL (pour les scores et les métadonnées des images)
- **Gestion de fichiers :** Upload d'images géré côté serveur.

## 🚀 Installation rapide
1. **Clonez le projet** : `git clone https://github.com/AinaCr/puzzleGame.git`
2. **Setup Server** : `cd server && npm install` (Configurez votre `.env`)
3. **Setup Client** : `cd client && npm install`
4. **Lancement** : `npm run dev` (client) et `node server.js` (server)
